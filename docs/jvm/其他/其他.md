# 4. 地址值

```java
int[] arr = {1,2,3,4,5};
System.out.println(arr);//[I@6d03e736

double[] arr2 = {1.1,2.2,3.3};
System.out.println(arr2);//[D@568db2f2
```

打印数组的时候，实际出现的是数组的地址值。

数组的地址值：就表示数组在内存中的位置。

以`[I@6d03e736`为例：

- `[` ：表示现在打印的是一个数组。

- `I`：表示现在打印的数组是int类型的。

- `@`：仅仅是一个间隔符号而已。

- `6d03e736`：就是数组在内存中真正的地址值。（十六进制的）

但是，我们习惯性会把`[I@6d03e736`这个整体称之为数组的地址值。
