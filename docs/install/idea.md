## 打开idea，填入激活码

如果在之前就打开了idea，那么显示 key is invalid。

本激活教程原理就是 阻止软件网络请求验证是否过期，其中的 有效期 、激活来源 均在配置文件（config/mymap.conf）中配置，补丁来源于jetbra.in。

我们可以在About查看激活有效期，有效期是多少无所谓，它是备用许可证，不会过期。
