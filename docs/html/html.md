最简单的结构
- 外部 html 标签
- 内部 head 和 body标签。

```html
<html>
    <head>
        <title>你好</title>
    </head>
    <body></body>
</html>
```


标签
- 不区分大小写。`<html>`, `<HTml>`
- 单引号、双引号都行。`<img src="1.png">`, `<img src='1.png'>` 

    html和js都是这样，不区分字符和字符串，故而引号更开放。从而可以直接 `var a = "<font color='red'>RED</font>"`
