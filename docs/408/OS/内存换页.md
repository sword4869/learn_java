1. 缓存，淘汰旧的，存入新的。
2. LRU，最近最少使用。使用，不仅put（更新、存新）算使用，get（访问）也算使用。
3. hashmap + 双链表
    - node存在key，因为淘汰结点时，map删除需要结点的key。
    - count和capacity来设置缓存大小
    ```
    put:
    ->map存在：[更新val] [移动到头]
    -> 不存在：[头插结点] [map入] [count++]
        -> 超容：[链表断尾] [map删除] [count--]

    get:
    -> map不存在：返回-1
    -> map存在：[移动到头] [返回val]
    ```
