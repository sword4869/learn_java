# 证书信任链

签发顺序：根证书→中间证书→服务器证书。

1、请求证书过程：收到服务器证书，里面记录着中间证书的CA名称，去CA请求中间证书……往上找到根证书，根证书是自签证书（操作系统里一般都会内置一些根证书）。

2、验证过程：拿着根证书的公钥去验证中间证书，中间证书可信，再用中间证书的公钥去验证服务器证书。
