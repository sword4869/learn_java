## null则赋值

```java
import java.util.Optional;

Integer a = null;
// a = new Integer(1);
// 非null用原值，null则用 orElse的值（类型要保持一致）
Integer result = Optional.ofNullable(a).orElse(-1);
System.out.println(result);
```
