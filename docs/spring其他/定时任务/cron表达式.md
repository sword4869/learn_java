```java
# 秒 分 时 日 月 星期几 [年]
```

```
            ┌──────────── 秒 (0 - 59)
            | ┌────────── 分 (0 - 59)
            | | ┌──────── 时 (0 - 23)
            | | | ┌────── 日 (1 - 31)
            | | | | ┌──── 月 (1 - 12) OR jan,feb,mar,apr ...
            | | | | | ┌── 星期几 (0 - 6, 星期天 = 0)(1-7, 星期一=1) OR sun,mon ... 
            | | | | | |
            * * * * * * 命令
星期几需要测试！！！！！！不同的软件设置不同
            
星号（*）：表示匹配任意值。例如，* 在分钟字段中表示每分钟都执行。
逗号（,）：用于分隔多个值。例如，1,3,5 在小时字段中表示 1 点、3 点和 5 点执行。
斜线（/）：用于指定间隔值。例如，*/5 在分钟字段中表示每 5 分钟执行一次。
连字符（-）：用于指定范围。例如，10-20 在日期字段中表示从 10 号到 20 号。


星期或日的特殊设置：一个为(?/L/x#n/1w/LW)，另一个必须为?
(?)：仅用于日期和星期几字段，表示不指定具体值。通常用于避免冲突。
(L)：即last，只能用于日和星期。
	`0 0 22 L * ?` 用在日则表示该月的最后一天
	`0 0 22 ? * L` 用在星期则表示星期六。
(x#n): 表示该月第n个星期x，仅用于星期。
	`0 0 22 ? * 6#2` 表示该月的第2个星期五
(1W)：即weekday，只能用于日，表示距离当月给定日期最近的工作日（周一到周五）
	`0 0 20 1W * ? *`，即当月距离1号最近的工作日那天触发，此时，如果1号就是工作日，那就这天触发，如果1号是周六，那就3号触发。
(LW): 只能用于日。表示当月的最后一个工作日触发事件
	`0 0 20 LW * ? *`
```
